# Mím Status

Check the current state of the knowledge Wellspring.

Run the following command to see pending reviews and knowledge status:

```bash
mim status
```

If there are pending reviews, run:

```bash
mim review
```

This launches the Bridge Guardian TUI where you can review and approve knowledge entries.
